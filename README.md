# مقرري+ (Muqarari+) 🎓🚀

**منصة تعليمية مبتكرة لسد الفجوة بين المقررات الأكاديمية والتطبيق العملي في سوق العمل.**
تم بناء هذا المشروع كجزء من المشاركة في الهاكاثون التقني.

## 🌟 فكرة المشروع
تهدف منصة "مقرري+" إلى ربط طلاب الجامعة بخبراء الصناعة والممارسين المعتمدين. تتيح المنصة للخبراء إضافة إثراءات وتطبيقات عملية (Real-world Applications) للمقررات الأكاديمية، مما يمنح الطلاب خارطة طريق واضحة لكيفية استخدام ما يدرسونه في بيئة العمل الحقيقية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **الخلفية (Backend):** Java 17, Spring Boot 3.2.4, Spring Security, Spring Data JPA
* **قاعدة البيانات (Database):** MySQL 8.x
* **الواجهة الأمامية (Frontend):** Thymeleaf, HTML5, CSS3, Bootstrap 5.3 (Bilingual RTL/LTR)
* **التشفير والأمان:** BCrypt Password Encoder

## ⚙️ كيفية تشغيل المشروع محلياً (للمطورين)
1. قم باستنساخ المشروع: `git clone [رابط المستودع]`
2. تأكد من تشغيل خادم `MySQL` لديك (عبر XAMPP أو غيره).
3. قم بإنشاء قاعدة بيانات فارغة باسم: `muqarari_plus`
4. افتح المشروع باستخدام `IntelliJ IDEA`.
5. قم بتشغيل الأمر: `mvn clean install`
6. قم بتشغيل ملف `MoqarariPlusApplication.java`.
